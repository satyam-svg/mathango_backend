Filter Chapters

Endpoint

GET {{base_url}}/api/v1/chapters

Parameters

  • page (current page)
  • limit (items per page)
  • status (e.g., "Not Started")
  • isWeakChapter (true/false)
  • subject (e.g., "Physics")
  • class (e.g., "Class 11")

Example

{{base_url}}/api/v1/chapters?page=1&limit=30&status=Not+Started&isWeakChapter=true&subject=Physics&class=Class+11

Get Chapter by ID

Endpoint

GET {{base_url}}/api/v1/chapters/:id

Example

{{base_url}}/api/v1/chapters/68405235d4f676494ee4fa77
